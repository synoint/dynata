# PHP Dynata API

This is a PHP library to work with [Dynata LTD](https://www.dynata.com/)
 public APIs:

* [Demand API](https://developers.dynata.com/demand-api/getting-started)


## Requirements

* PHP 7.1+
* [Guzzle](http://guzzlephp.org)

## Installation

```bash 
composer require syno/dynata
```
